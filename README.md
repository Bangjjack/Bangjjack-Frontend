## 💡 프로젝트 배경
- 기숙사 룸메이트 AI 매칭 및 모집글 탐색 서비스 제공
- 생활습관 체크리스트를 하나의 서비스에서 간편하게 작성
- AI 기반 생활습관 분석을 통해 나와 잘 맞는 룸메이트 추천
- 공동생활 가이드를 통해 체크포인트, 대화 주제, 생활 규칙 설정 지원으로 원활한 기숙사 생활 지원
